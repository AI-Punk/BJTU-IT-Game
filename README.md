# BJTU-IT-Game
一个预计用在2020年第一学期的全校性的闯关游戏
# 规则
1. 每一关通过后会获得进入下一关的URL, URL采用加密(或者巨长)的方式来避免被爆搜
2. 在适当的关卡中插入KEY值(数量待定),每一个key值可以兑换一份奖励
# 关卡描述
1. 999 取整 (1024) 
3. 字符加密
4. 注释中写下一个页面的URL
5. 透明文字和不显示文字
6. setTimeout
7. 网络爬虫
8. 双生文件
9. 文件处理
10. base64藏值
11. 图片藏值（类似windows下copy方法融合)
11. 字符提取
12. hidden输入框藏值
